## 🌟 About The Project

Welcome to the **StackByUjjwal Post Generator**! Creating visually appealing posts for tech pages, coding accounts, and motivational channels can be time-consuming. This web-based tool solves that by allowing you to generate high-quality, customized images in seconds. 

No backend setup, no heavy software—just pure frontend magic using `html2canvas`.

### 🔗 **[Click Here to Explore the Live Project](https://stackbyujjwal.github.io/StackByUjjwal-post-Generators/)**

---

## ✨ Features

- 🎨 **Multiple Custom Templates:** Choose from Terminal, Quiz, Quotes, and Soulful designs.
- ⚡ **Real-time Editing:** See your changes instantly as you type.
- 🖼️ **HQ Export:** Downloads optimized `1080x1350` High-Quality PNGs directly to your device.
- 📱 **Fully Responsive:** Works seamlessly on desktop and mobile.
- 🔒 **Privacy First:** 100% Client-side rendering. No data is sent to any server.

## 🛠️ Built With

- **HTML5** - For robust structure.
- **CSS3** - For modern styling, flexbox, grid, and CSS variables.
- **Vanilla JavaScript** - For DOM manipulation and logic.
- **[html2canvas](https://html2canvas.hertzen.com/)** - The magic library that converts our DOM elements into downloadable images.
- **FontAwesome** - For sleek social icons.
